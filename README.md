# KnowHub Ethiopia 🇪🇹

Welcome to **KnowHub Ethiopia**, a premium, modern, and highly responsive web application designed for both normal desktop/mobile browsers and as a **Telegram Mini App**. 

Our mission is to empower Ethiopian students by providing free, centralized access to schools, scholarships, test preparation guides, study resources, and academic opportunities.

---

## ✨ Features

- **📱 Dual Mode Support**: Smoothly integrates with the Telegram Mini App SDK (with graceful fallback on normal web browsers).
- **🎨 Modern Premium UI/UX**: Designed with vibrant, harmonious color schemes, glassmorphism elements, custom scrollbars, and high-performance micro-animations.
- **⚡ Built for Speed**: Powered by Vite and React 19 for instantaneous page loads and lightning-fast navigation.
- **📊 Real-time Stats**: Track 45+ Schools, 120+ Scholarships, 80+ Key Opportunities, and 200+ Free Courses.
- **📚 Curated Resources**: Comprehensive sections for IELTS/TOEFL preparation, study materials, and direct links to our upcoming AI Tutor.
- **🎯 100% Free**: Built entirely to support student journeys without registrations or hidden fees.

---

## 🛠️ Technology Stack

- **Core Framework**: [React 19](https://react.dev/)
- **Bundler & Dev Server**: [Vite 8](https://vite.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Styling**: Pure vanilla CSS with custom variables, smooth transitions, and modular layout rules.
- **Platform Integration**: Telegram WebApp SDK

## 🚀 Getting Started

### 📋 Prerequisites

Ensure you have **Node.js** (v18 or higher) and **npm** installed on your machine.

### ⚙️ Installation

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/AbdurahmanHussein/KnowHub_Ethiopia.git
   cd KnowHub_Ethiopia
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Start the Development Server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:5173](http://localhost:5173) in your browser.

4. **Build for Production**:
   ```bash
   npm run build
   ```

## 🌐 Deployment to Vercel

This project is fully configured and ready to be deployed to Vercel with zero-configuration:

1. Connect your GitHub account to [Vercel](https://vercel.com).
2. Click **New Project** and import `KnowHub_Ethiopia`.
3. Set the Framework Preset to **Vite**.
4. Click **Deploy**! 🚀

A `vercel.json` file is already included in the root to ensure Single Page Application (SPA) routing works perfectly without 404 errors.
